# To-Do List Application (Assignment 01)
**Student Name:** Mariam Ashraf Hassan Mohamed

## Project Description
A React Native mobile application built with Expo for managing daily tasks. The app features a coordinated color scheme, custom fonts, and a dual-scroll system for active and completed tasks.

## Live Demo
You can try the live application on Expo Snack here:
https://snack.expo.dev/@mariam_ashraf26/to-do-list-

## Features
* **Logic:** Implemented using React Hooks (useState, useEffect).
* **Components:** Custom TextInput, TouchableOpacity buttons, and FlatLists.
* **Scrolling:** Optimized Dual Independent Scroll for Android and Web.
* **Design:** Professional Emerald Green palette with "Delius-Regular" custom font.
